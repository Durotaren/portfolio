# Portfolio

This project is a personal portfolio built to showcase my work, skills, and development experience. It focuses on modern frontend practices, performance, and maintainable code.

---

## Overview

The application presents selected projects with descriptions, source code links, and live demos. It includes a theme system and subtle animations to enhance the user experience while keeping the interface fast and responsive.

---

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- Vite

---

## Key Features

- Light and dark mode support
- Smooth UI animations
- Interactive project showcase
- Optimized performance

---

## Purpose

The goal of this project is to demonstrate practical frontend skills, including component-based architecture, state management, and modern styling techniques.
